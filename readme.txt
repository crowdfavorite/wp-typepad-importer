== Description ==

Import posts, comments, and images from a TypePad blog.

Attempts to intelligently download the full size images from TypePad asset
links, strip out links which pop up a fullsize version of the image, and
rewrite other links to point to the appropriate post (where possible) or to the
same place in the new blog otherwise.

== Installation ==

1. Upload the `typepad-importer` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the Tools -> Import screen, Click on TypePad
